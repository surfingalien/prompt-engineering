# design-system-auto-prompt.ts

Source: `apps/web/src/design-system-auto-prompt.ts`

Defines the fixed prefix string that Open Design prepends when it
auto-generates a project as a "design system workspace" (i.e. when a user
sets up a project from design-system setup sources rather than a freeform
prompt). `isDesignSystemWorkspacePrompt` is a detector used elsewhere in the
app to recognize messages that were generated by this auto-prompt flow (so UI
can special-case them), by checking whether a message's content starts with
the prefix below.

## DESIGN_SYSTEM_WORKSPACE_PROMPT_PREFIX

The literal prompt-prefix text injected as the start of the auto-generated
project-creation message:

```
Create this project as a complete Open Design design system workspace.
```

## Accompanying UI strings (not prompt text sent to the model, included for completeness)

- Display title: `Creating design system workspace`
- Display description: `Open Design is using the setup sources to generate this project.`
